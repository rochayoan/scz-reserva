# SCZ-RESERVA

## Visión del proyecto

SCZ-RESERVA es una plataforma SaaS de reservas deportivas para Santa Cruz de la Sierra.

El objetivo es permitir que los usuarios encuentren y reserven canchas deportivas de forma rápida, mientras los propietarios administran sus complejos desde un panel web moderno.

El MVP actual es una demostración funcional visual.

La visión futura es convertirlo en una plataforma SaaS multi-tenant donde los dueños de complejos deportivos paguen una suscripción mensual para administrar sus canchas.
